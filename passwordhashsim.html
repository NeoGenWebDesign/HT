What is Password Hashing?

Password Hashing is a one-way mathematical function that converts a variable-length input (the password) into a fixed-length string of characters (the hash). This is crucial for security because it means the plaintext password is **never stored**, only its hash. The process shown below uses a cryptographic hash function (SHA-256) and a unique **Salt** for better security against pre-computed attacks (Rainbow Tables).
